```javascript
![](https://...)
```

```javascript
![](./images/pic.png)
```

### 添加多说评论功能
多说是一个开源的，提供分享到社交平台并提供给使用者管理评论内容的这么一个插件，说实话很好用，配上我blog的mau主题显得很不错;
添加方法：
```javascript
在多说上申请自己的账号和域名后;
在主题的config.yml文件的duoshuo:栏中加上自己域名简称就行;
用多说的unstable版本，原因：最新稳定版会导致分享显示的社交平台为undefined;
```
